# 快速开始

`react-antd-xform` 是一个基于 MobX 和 Ant Design 的动态表单解决方案，提供 JSON Schema 驱动、表单联动和复杂校验等高级功能

## 📦 安装

选择你喜欢的包管理器进行安装：

```bash
# npm
npm install react-antd-xform --save

# yarn
yarn add react-antd-xform

# pnpm
pnpm add react-antd-xform
